# Word Forest

Word Forest is a simple anagram game for kids.

Players see a scrambled word and try to guess the correct word. The game includes hints, score tracking, and short celebrations for correct answers.

## Language

The game is made for a Norwegian audience.
The user interface text is in Norwegian.
The word list is also Norwegian.

## Files

- `index.html` contains the page structure
- `style.css` contains the design
- `app.js` contains the game logic and word list

## Use and editing

Anyone can download, use, and edit this project for non-commercial purposes.

If you want to change the words, text, or design, you can edit the files directly.
